# FailTrace — API Failure Simulation System

A system designed to simulate real-world API failures and analyze system behavior.

## Why this project

To understand how systems behave under failure conditions and improve debugging workflows.

## Features

- Simulated API failures (timeouts, errors, auth issues)
- Structured logging
- Debugging workflows
- Health check endpoints

## Tech Stack

Node.js, TypeScript

## What this demonstrates

- Debugging skills
- Logging and monitoring
- System reliability thinking
